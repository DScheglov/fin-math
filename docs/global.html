ipmt - Returns the interest payment for a given period for an investment based on periodic, constant payments and a constant interest rate
Parameters:
Name Type Attributes Default Description
rate Number the rate should be used to accrue interests for one period
period Number specifies the period and must be in the range 1 to `periods`
periods Number number of periods
start_amount Number the start amount of the loan
fv Number <optional>
 0 The future value, or a cash balance you want to attain after the last payment is made.
pmt_type Number <optional>
 0 if 0 or ommitted the function considers payment at the end of period, otherwise at the beggining

pv(rate, periods, pmtopt, fvopt, pmt_typeopt) → {Number}

pv - calculates the present value of a loan or an investment, based on a constant interest rate. You can use PV with either periodic, constant payments (such as a mortgage or other loan), or a future value that's your investment goal.
Parameters:
Name Type Attributes Default Description
rate Number The interest rate per period.
periods Number The total number of payment periods in an annuity
pmt Number <optional>
 0 The payment made each period and cannot change over the life of the annuity.
fv Number <optional>
 0 The future value, or a cash balance you want to attain after the last payment is made.

WARNING: The MS Excel considers parameter `fv` as final payment that reduces loan balance to zero. It means the sign of the `fv` should be the same as the sign of payments.
pmt_type Number <optional>
 0 The number 0 or 1 and indicates when payments are due (0 - end of period).
